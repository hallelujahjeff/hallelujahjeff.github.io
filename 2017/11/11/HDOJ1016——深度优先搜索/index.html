HDOJ1016素数环——深度优先搜索初探

HDOJ1016

  • 问题描述

  A ring is compose of n circles as shown in diagram.Put natural number 1, 2, …, n into each circleseparately, and the sum of numbers in two adjacent circles should be a prime.Note: the number of first circle should always be 1.

input

n (0 < n < 20)

output

  • 分析

  看了下Discuss板块，了解到这道题应该用DFS(深度优先搜索)的算法思想来做，首先去知乎了解了一下
搜索思想——DFS & BFS（基础基础篇）

  深度优先搜索的步骤分为 1.递归下去 2.回溯上来。顾名思义，深度优先，则是以深度为准则，先一条路走到底，直到达到目标。这里称之为递归下去。
  否则既没有达到目标又无路可走了，那么则退回到上一步的状态，走其他路。这便是回溯上来。

  • 实现

经过两天半的努力，把素数环的代码刚出来了，主要还是这个DFS的实现吧。
首先，判断成立的条件有

  1. n个数全部用到
  2. 每两个数之间的和为素数
  3. 首位的和也为素数

定义两个数组，一个为a[n]，另一个为save[n]，分别用于记录数字的使用情况和当前路径。

一个函数scan，用来检测所有数是否都被用到。

一个函数prime，用来检测两数相加是否为素数。

定义一个location，记录当前位置的数组下标
DFS实现的过程是：程序从1开始遍历，如果当前位置的值(location+1)和下一位置的值(i+1)的和为素数，且下一位置的值未被标记，就调用下去。如果遍历完后，达不到成立的条件，则回溯到上一层，继续遍历。

完整代码

/*HDOJ1016 素数环问题 DFS深度优先搜索训练*/
#include<stdio.h>
#include<malloc.h>

int count; //记录save数组当前位置
int Case = 1;
int main()
{
    void dfs(int *,int *,int,int);
    int n,i;
    while(scanf("%d",&n) != EOF )
    {
        if(n == 1)  //若n为特殊情况，直接打表输出
        {
            printf("Case %d:\n1\n\n",Case);
            Case++;
            continue;
        }
        int *a=(int *)malloc(n*sizeof(int)); //定义动态数组 相当于int a[n]
        int *save=(int *)malloc(n*sizeof(int));  //定义save数组，记录路径
        count = 0;  //save数组位置初始化
        for(i=0;i<n;i++)
        {
            if(i==0) *a = 1;
            else *(a+i) = 0;
        }
        for(i=0;i<n;i++)  //初始化save数组，用于输出路径
        {
            if(i==0) *save = 1; //路径的开头必为1
            else *(save+i) = 0;
        }
        printf("Case %d:\n",Case);
        Case++;
        dfs(a,save,0,n);
        printf("\n");
    }
}

/*检测当前位置和下一位置的和是否为素数*/
int prime(int location,int next)
{
    int sum = location + next;
    if (sum==3||sum==5||sum==7||sum==11||sum==13||sum==17||sum==19||sum==23||sum==29||sum==31||sum==37)
        return 1;
    else
        return 0;
}

/*检测是否填满n个数字*/
int scan(int *a,int n)
{
    for(int i=0; i<n; i++)
    {
        if(*(a+i) == 0) return 0;
    }
    return 1;
}

/*深度优先搜索，遍历每一种情况*/
void dfs(int *a, int *save, int location,int n)
{
    if(scan(a,n) == 1 && prime(location+1,1)) //如果n个数填满，且头和尾相加为素数，则达成目标
    {
        for(int i=0;i<n-1;i++)
        {
            printf("%d ",*(save+i)); //打印路径
        }
        printf("%d",*(save+n-1));  //控制格式，末尾无空格
        printf("\n");
        return;
    }
    for(int i=1;i <n; i++)
    {
        if(*(a+i) != 1 && prime(location+1,i+1)) //如果下一位置未标记，且相加为素数
        {
            count++;  //save数组位置后移
            *(a+i) = 1;  //标记下一位置
            *(save+count) = i+1;  //save数组赋值
            dfs(a,save,i,n);  //调用下一位置的dfs
            *(a+i) = 0;   //如果dfs无结果，则回溯到上一位置，且重置标记
            *(save+count) = 0;  //回溯，重置save数组
            count --;  //save数组前移
        }

    }

}
